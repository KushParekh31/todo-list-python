# To-Do List (Python)

A simple command-line To-Do List application built with Python.

## Features

* Add tasks
* View tasks
* Mark tasks as completed
* Delete tasks
* Save tasks permanently using JSON
* Track creation and completion timestamps

## Requirements

* Python 3.x

## How to Run

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/todo-list-python.git
```

Move into the project directory:

```bash
cd todo-list-python
```

Run the program:

```bash
python main.py
```

## Example

```text
===== TO-DO LIST =====
1. Add Task
2. View Tasks
3. Complete Task
4. Delete Task
5. Exit
```

* Python
